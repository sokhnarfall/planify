# API Documentation

## Server
http://localhost:3000/

1. Path: api.get('/')
- Directs the user to login or register

2. Path api.get('/homepage')
- Renders the homepage and displays user's tasks

"/login"
- Renders the login page

"/signup"
- Renders the registration page

### POST

"/login"
- Handles user login

"/signup"
- Handles user registration
